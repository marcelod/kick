# What is Kick

Kick is a project to start a site with some features using Codeigniter, HMVC, IonAuth, Composer, Bower, etc.

But has a lot to be done yet.

This is a fork [CI-AdminLTE](https://github.com/domProjects/CI-AdminLTE)
